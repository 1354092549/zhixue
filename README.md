## Introduction
这是一个由第三方提供的轻巧的智学网PC客户端（具有Windows/Linux/Mac版）   
方便电脑端查分

## Build

```bash
git clone https://github.com/1354092549/zhixue
cd zhixue
npm install
bower install
npm run package
```

## Run
The output file is `OutApp/zhixue-{platform}-{arch}/zhixue.exe`  
eg. `OutApp/zhixue-win32-x64/zhixue.exe`
